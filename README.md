# FitCore Gym – Fitness Center Website (UI Template)

FitCore Gym is a modern, energetic, and responsive website UI designed specifically for fitness centers, gyms, and personal trainers. Built entirely using **pure HTML5 and CSS3, this static web template provides a professional online presence layout to showcase gym facilities, training programs, membership plans, and expert coaches. It is ideal for developers looking for a ready-to-use, responsive fitness template to integrate with their backend systems or use as a landing page.

## 💻 Used Technologies

* **HTML5** (Semantic structuring)
* **CSS3** (Custom properties, flexbox/grid, pure CSS transitions, responsive media queries)
* **Font Awesome** (Icons)
* **Google Fonts**
* **GitHub Pages** (Hosting)

*(Note: This project is built completely without JavaScript, focusing purely on structural design and CSS styling.)*

---

## ✨ Key Features

* Energetic and modern user interface tailored for the fitness industry
* Responsive navigation bar layout
* Impactful hero section design with Call-to-Action (CTA) buttons
* Dedicated section layouts for fitness classes, services, and schedules
* Trainers and coaches showcase design with social links
* Clear and structured membership pricing plans UI
* Smooth pure CSS scrolling and hover effects
* Fully responsive design across all devices (Mobile-first approach)

## 📥 Installation Instructions for Local Setup

1. **Clone the repository:**

```bash
git clone https://github.com/qanawy21/FitCore-Gym.git

```

2. **Navigate to the project directory:**

```bash
cd FitCore-Gym

```

3. **Run the project:**

open `index.html` in your browser

> No dependencies, build tools, or JavaScript are required—just open the file directly!
